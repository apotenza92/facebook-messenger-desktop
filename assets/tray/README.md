# System Tray Icons

Place your system tray icons here:

- `icon.ico` - Windows tray icon (16x16 or 32x32)
- `iconTemplate.png` - macOS tray icon (22x22, should be a template image with transparency)
- `icon-rounded.png` - Linux tray icon (24x24, colored rounded Messenger icon)

Note: `icon.png` is kept as a fallback but the app now uses `icon-rounded.png` for Linux.
