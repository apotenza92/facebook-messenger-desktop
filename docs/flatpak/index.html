Facebook Messenger Desktop - Flatpak

This is a self-hosted Flatpak repository for Facebook Messenger Desktop.

Installation

# Add the repository
flatpak remote-add --if-not-exists messenger-desktop https://apotenza92.github.io/facebook-messenger-desktop/flatpak/repo --gpg-import=https://apotenza92.github.io/facebook-messenger-desktop/flatpak/repo/flatpak-repo.gpg

# Install the app
flatpak install messenger-desktop io.github.apotenza92.messenger

Updates

flatpak update io.github.apotenza92.messenger
